各类工具汇总

  • 工具
收藏品

收集开发学习过程中使用的各种工具，持续更新。

About windows

  • clover - 重度chrome用户，给windows资源管理器加上chrome标签
  • Everything - 快速找到windows上所有的一切，再也不用去繁琐的一层一层翻文件夹
  • 其他的看这里有哪些工具或软件堪称神器 - 知乎

About php

  • phpstorm - jetbrains大法好，用过的最舒适的IDE，界面酷炫代码提示友好集成了很多的工具，jetbrains家的IDE每一款都是业界良心，唯一不足是费用略贵。
  • Laravel - 喜爱优雅的代码吗？我们也一样；为网页艺术家创造的 PHP 框架
  • Lumen - 为速度而生的Laravel框架（我实在是喜欢laravel优雅的语法），基于laravel的微框架，laravel的功能加上极致的速度。
  • wsdl-creator - php wsdl生成库

About python

  • pycharm - 一样jetbrains大法好
  • Scrapy - 高效的python爬虫框架，一个高效的爬虫可能不到100行代码，scrapy-example使用scrapy爬各种网站的案例
  • Flask - python web 微框架，核心部分只提供最基础的功能，其他的部分都可以扩展，即插即拔的组件式开发，非常灵活多变
  • xlwings - 在Excel中执行python，在python中执行Excel

About javascript

  • Bootstrap
  • bootstrap-theme-slim - 纤细风格的bootstrap主题
  • B-JUI - Bootstrap for dwz富客户端框架
  • Simditor - 简单快速的富文本编辑器
  • markdown-here - markdown写邮件

About 大神博客

  • overtrue - 安正超博客
